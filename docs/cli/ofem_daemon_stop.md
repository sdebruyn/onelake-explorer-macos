## ofem daemon stop

Stop the daemon

### Synopsis

Tell launchd to unload the OFEM daemon. The service will stay
unloaded until you run 'ofem daemon start' or log in again. To
permanently remove it, use 'ofem daemon uninstall'.

```
ofem daemon stop [flags]
```
