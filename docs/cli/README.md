# CLI reference (generated)

This directory is generated by `make docs-cli` from the cobra command
tree in `cmd/ofem/cli/`. Do not edit files here by hand — every
`ofem*.md` file is wiped and rewritten on each regeneration run.

To update the published reference:

1. Change the cobra command, flag, or short/long description in
   `cmd/ofem/cli/`.
2. Run `make docs-cli` at the repo root.
3. Commit the regenerated `docs/cli/ofem*.md` files together with the
   matching nav block update in `zensical.toml`.

CI fails the build if the committed files drift from the cobra tree,
so the published reference can never go stale.

This `README.md` is not part of the generated set and is preserved
across regeneration runs (the Makefile's `find -delete` is scoped to
`ofem*.md`).
