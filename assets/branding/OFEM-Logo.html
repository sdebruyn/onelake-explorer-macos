Usage notes

  • The primary mark is ofem-mark.svg (200×200 viewBox, gradient-filled). Scales cleanly from 16 px up.
  • Wordmark is set in Inter Tight 600, tracking -0.02em. Accent . uses #0078D4 on light surfaces and #5BC5FF on dark.
  • Use ofem-mark-mono-* for any single-color context (favicons under 16 px, print, embossing). Opacity 1.00 / 0.72 / 0.46 preserves drop hierarchy without color.
  • ofem-icon.svg is the full macOS squircle treatment at 1024×1024 — feed to iconutil for the .icns bundle. Rendered into 16/32/64/128/256/512/1024 PNGs at export time.
  • Minimum clear space around the mark = the height of the small accent drop.
  • Do not recolor the gradient stops, do not flip the gradient direction, do not separate the three drops. If you need a single drop variant, request a derivative; do not edit the source.
